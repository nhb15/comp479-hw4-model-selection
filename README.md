# comp479-hw4-model-selection

These files can be run in the following way:


1. Get into the src/ folder with cd src
2. To run the NFoldClassifier, type `python3 NFoldClassifier.py`
   1. To see the results, view the console.
3. To run the Grid Search, type `python3 GridSearchImplementation.py`
   1. To see the results, you may use the console or the `gridresults.csv` with better formatting
